# CHZZK Replay Repeat

치지직(CHZZK) 다시보기에서 특정 구간을 반복 재생할 수 있는 Chrome 확장 프로그램입니다.

## 설치

Chrome Web Store에서 설치할 수 있습니다:

[CHZZK Replay Repeat 설치하기](https://chromewebstore.google.com/detail/chzzk-replay-repeat/jfaajoijcapjeojphmppnmpeglpcmfma?authuser=0&hl=en)

## 주요 기능

- **구간 반복 재생**: 비디오의 시작 지점과 끝 지점을 설정하여 원하는 구간을 반복 재생
- **드래그 가능한 UI**: 오버레이 패널을 원하는 위치로 자유롭게 이동
- **최소화 기능**: 패널을 최소화하여 화면을 깔끔하게 유지
- **직관적인 컨트롤**: 버튼 클릭 또는 직접 시간 입력으로 구간 설정

## 사용 방법

1. 치지직 다시보기 페이지에 접속합니다
2. 비디오가 로드되면 우측 상단에 구간 반복 패널이 자동으로 표시됩니다
3. 반복하고 싶은 구간의 시작 지점에서 **"시작 지점 설정"** 버튼을 클릭합니다
4. 끝 지점에서 **"끝 지점 설정"** 버튼을 클릭합니다
5. **"반복 시작"** 버튼을 클릭하여 구간 반복을 활성화합니다
6. 시간 입력 필드에서 직접 시간(HH:MM:SS)을 입력하여 정확한 구간을 설정할 수도 있습니다

### 추가 기능

- **패널 이동**: 패널 상단을 드래그하여 원하는 위치로 이동
- **최소화**: 상단의 **−** 버튼을 클릭하여 패널 최소화/복원
- **닫기**: 상단의 **✕** 버튼을 클릭하여 패널 숨기기
